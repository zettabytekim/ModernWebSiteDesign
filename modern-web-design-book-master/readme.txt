▽grid-guide.css에 대해서

grid-guide.css는 박스의 구조를 이해하기 쉽게 표시하기 위해 만들어진 스타일 시트입니다.

- 클래스 이름이 "box"로 시작하는 요소를 테두리로 감싸서, 박스의 구조를 시각화합니다.
- 박스의 계층에 따라서 색상이 변경됩니다.
     최상위 계층: 붉은색
     자식 계층: 노란색
     손자 계층: 파란색
- 박스 내부에 "BOX1"처럼 문자를 넣으면, 높이가 200픽셀로 고정되어 표시됩니다.
- 박스 내부에 <div> 등으로 마크업한 요소를 넣으면, 200픽셀의 높이가 해제되며, 내부 요소의 높이에 맞춰집니다.
